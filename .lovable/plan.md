

# Plan: Bombas Total, Invoice PDF + Duplicate, Volunteer Cancel

Three independent features, each scoped to the files and tables that need to change.

---

## 1. Bombas Total Pairs line on invoices

Add a "Bombas Pairs" line that sums quantity for any invoice item whose item name contains "Bombas" (case-insensitive). It appears just below the existing "Total Pairs" line.

**Where it shows up:**
- **Invoice detail dialog** -- new line below "Total Pairs" inside `InvoiceDetailDialog.tsx`
- **Invoice list table** -- new "Bombas" column to the right of the existing "Pairs" column in `InvoicesTable.tsx`
- **Printable invoice template** -- new "Bombas Pairs" summary row below the items table inside `InvoiceTemplate.tsx`

**Logic:** `bombasPairs = sum of item.quantity where item.item_name.toLowerCase().includes("bombas")`. Only displayed when value > 0 in the print template; always shown (even as 0) in the dialog and table for consistency.

**Files touched:**
- `src/components/InvoiceDetailDialog.tsx`
- `src/components/InvoicesTable.tsx`
- `src/components/InvoiceTemplate.tsx`

No backend changes.

## 2. Download invoice as PDF + Duplicate invoice to a different partner

### 2a. PDF download

Add a "Download PDF" button next to the existing "Print" button in the invoice detail dialog. Generate the PDF client-side from the existing `InvoiceTemplate` markup using `html2canvas` + `jspdf` (lightweight, no backend). Filename: `Invoice-{invoice_number}.pdf`.

### 2b. Duplicate invoice to different customer

Add a "Duplicate" button in the invoice detail dialog (visible for any status). Clicking opens a small dialog `DuplicateInvoiceDialog.tsx` with:
- A customer picker (searchable dropdown, same pattern as `CreateInvoiceDialog`)
- Optional new due date
- "Create Duplicate" button

On submit:
- Insert a new row in `invoices` (status `draft`, new invoice number auto-generated by existing `set_invoice_number` trigger, selected customer, today's invoice date, copied notes)
- Insert copies of all `invoice_items` rows linked to the new invoice id
- Show toast and open the newly created duplicate

**Files touched:**
- `src/components/InvoiceDetailDialog.tsx` -- add Download PDF + Duplicate buttons
- `src/components/DuplicateInvoiceDialog.tsx` -- new file
- `package.json` -- add `jspdf` and `html2canvas`

No backend/schema changes (existing RLS allows insert).

## 3. Cancel a volunteer signup (free up slot)

Add a "Cancel" action to each row in the Volunteer Signups admin table. Clicking opens a confirmation dialog ("Cancel signup for {name}? This frees up {quantity} slot(s)."). Confirming deletes the row from `volunteer_signups`.

The existing `update_event_slots` DB trigger already decrements `slots_filled` on DELETE -- no app-side slot math needed.

**Schema change required:** the `volunteer_signups` table currently has no DELETE RLS policy. Add one via migration:

```sql
CREATE POLICY "Allow public to delete volunteer signups"
ON public.volunteer_signups FOR DELETE
USING (true);
```

**Files touched:**
- `src/pages/VolunteerSignups.tsx` -- add Actions column with Cancel button + AlertDialog confirmation, refetch on success
- One migration to add the DELETE policy
